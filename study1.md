## torch.max 
用于测试函数

不指定维度，返回最大的
torch.max(input) -> Tensor
```python
a = torch.randn(1, 3)
a
>>tensor([[-1.9542,  0.2564, -2.3612]])
torch.max(a)
>>tensor(0.2564)


a = torch.randn(3, 3)
a
>>tensor([[-1.7755,  1.1617,  1.0005],
        [ 0.3386,  0.2030, -1.0289],
        [-0.2869,  1.4056,  1.5640]])

指定维度之后，返回的是指定维度对应位置最大的那个值
指定dim=0之后，得到的结果是维度为0的“每一行”对应位置求最大的那个值
torch.max(a, 0)        
>>torch.return_types.max(
values=tensor([0.3386, 1.4056, 1.5640]),
indices=tensor([1, 2, 2]))


a = torch.randn(2, 3, 4)
a
>>tensor([[[ 0.8409, -0.2519, -0.6617, -0.6825],
         [ 2.0912, -0.5912, -0.1007, -1.2286],
         [ 0.2420, -0.6835, -0.2634, -0.2248]],

        [[ 0.1482, -0.1316,  1.0710,  1.8814],
         [ 0.7637, -0.9082, -0.4977,  0.5236],
         [ 0.5750, -1.2943,  0.9164, -0.6118]]])

torch.max(a, -1, keepdim=True)        
>>torch.return_types.max(
    values=tensor([[[0.8409],
            [2.0912],
            [0.2420]],

            [[1.8814],
            [0.7637],
            [0.9164]]]),
    indices=tensor([[[0],
            [0],
            [0]],

            [[3],
            [0],
            [2]]]))
```
